# CV sweisgerber

> [!IMPORTANT]
> Rendered page: https://sweisgerber.github.io

-----

## Usage

```shell
pipenv install
pipenv shell
#
mkdocs new ./
#
mkdocs serve
#
mkdocs build
#
pipenv upgrade
```

## Usage - Generate PDF

```shell
MKDOCS_CV_BUILD_PDF=1 mkdocs build
MKDOCS_ENABLE_PDF_EXPORT=1 mkdocs build

```

## Update `data/` files

```shell
./tools/generate-metadata.py
```
